# ⚽ FIFA 2020 Data Analysis

This project performs Exploratory Data Analysis (EDA) on the FIFA 2020 player dataset using Python. The notebook explores player statistics, ratings, positions, wages, clubs, nationalities, and other attributes to gain meaningful insights from the data.

---

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📊 Analysis Performed

- Data Cleaning
- Handling Missing Values
- Descriptive Statistics
- Player Rating Analysis
- Club-wise Analysis
- Nationality Analysis
- Position Distribution
- Salary and Wage Analysis
- Data Visualization

---

## 📈 Visualizations

The notebook includes various charts such as:

- Bar Charts
- Histograms
- Scatter Plots
- Count Plots
- Box Plots
- Heatmaps

---

## 🎯 Project Objectives

- Understand player performance.
- Identify top-rated players.
- Compare clubs and nationalities.
- Visualize player statistics.
- Practice Exploratory Data Analysis (EDA).
